1. Variables

In C language, variables are used to store data values of different types. Variables act as containers for holding data that can be manipulated by the program during execution. Each variable in C must have a specific data type that defines the kind of value it can store (e.g., integer, float, character).
Basic Syntax:
data_type variable_name = value;
data_type: Type of data the variable will store (e.g., int, float, char).
variable_name: The name of the variable.
value (optional): Initial value assigned to the variable

2.Data Types

In C, data types define the type of data that a variable can hold and the operations that can be performed on it. C has several basic data types:
Integer types:
int: Used for storing whole numbers, typically 4 bytes in size.
short: A smaller integer type (usually 2 bytes).
long: A larger integer type (often 4 or 8 bytes).
Floating-point types:
float: Used for single-precision floating-point numbers (decimals).
double: Double-precision floating-point numbers, offering greater accuracy.
Character type:
char: Used for storing single characters (1 byte).
Void type:
void: Used to indicate the absence of a value, commonly in functions that do not return a value.
Additionally, there are modifiers like unsigned and signed to control whether variables can store negative values. Understanding these types is essential for memory-efficient and effective programming in C.

3. Operators

In C, operators are symbols that perform operations on variables and values. C provides several types of operators:
Arithmetic operators:
Used for mathematical operations like addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).
Relational operators:
Compare values and return true or false, such as greater than (>), less than (<), equal to (==), and not equal to (!=).
Logical operators:
Used for logical conditions, like AND (&&), OR (||), and NOT (!).
Bitwise operators:
Perform bit-level operations, including AND (&), OR (|), XOR (^), and bit shifts (<<, >>).
Assignment operators:
Assign values to variables, like the simple assignment (=) and compound assignments (+=, -=, etc.).

Conditional (ternary) operator:

These operators form the core of manipulating data and making decisions in C programs.
